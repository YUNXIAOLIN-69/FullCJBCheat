# FullCJBCheat 完整CJB手柄作弊菜单
适配：星露谷1.6 + SMAPI 4.0+
安装方法：
1. 安装SMAPI加载器
2. 将整个FullCJBCheat文件夹放入 Stardew Valley/Mods
3. dotnet build 编译生成dll
4. 使用SMAPI启动游戏

操作说明：
键盘：P 打开/关闭作弊菜单
手柄：
LT = 切换上一个标签页
RT = 切换下一个标签页
十字键上下 = 选择功能（自动吸附高亮整行）
十字键左右 = 调节滑块/切换开关
A = 执行按钮功能
B = 关闭菜单

特色：
1. 全部功能单行独立，无多控件拥挤
2. 数值调节统一条状滑块
3. 传送标签页置顶联机玩家传送，多人一键瞬移队友
4. 完整复刻原版CJB Cheats Menu全部功能
5. 所有设置自动保存到config.json
